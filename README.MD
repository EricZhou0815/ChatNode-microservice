#### works on node version > 12

### Tech-stack: 

#### Typescript (Node, React), GraphQL, Docker, MySQL

### Structure:

#### chat-app (Frontend) -> api-Gateway (GraphQL) -> (users-service -> users-service-db | chat-service -> chat-service-db)
